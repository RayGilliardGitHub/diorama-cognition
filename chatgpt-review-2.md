## New Insight: Periodic Stimulus Confound

The 5 prompts repeating every 5 turns means chemistry is learning the **test schedule**, not reacting to emotional significance. This is a major confound we missed — the chemistry trajectory may be driven by prompt frequency, not content.

## Strongest recommendations

1. **Compare Phase 2 − Phase 1b** (not Phase 2 − baseline). Phase 1b is the real null hypothesis.
2. **Run 200-500 diverse prompts** — the periodic stimulus confound ruins the current data
3. **Mechanistic gating** — enforce token budgets/visibility per gate value, don't ask LLM to self-report
4. **Chemistry needs habituation, prediction error, opponent processes** — additive deltas are wrong
5. **Information knockout experiment** — randomize or contradict node outputs, measure if synthesizer ignores them
